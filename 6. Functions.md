After nama api link potu send panna apparam

Params - API ku ethana parameters irukku na inga kudukalam or athuvey direct ah eduthukum

Authorization - Nama API ku ithu tha thevai

{
Authentication = Who you are - nama yaaru nu validation kuduthu enter aagurathu

Authorization = What you can do - Nama inga enna 
}

So intha case la Authorization tha unga thevai padum

Headers - nama enna mathiri header send pana porom nu inga kudukalam

Body - namakku GET ku thavai padathu, POST and PUT ku tha thevai padum

Scripts - Postman la irukura intha function nama inga JavaScript eluthikalam, ithu la rendu irukku

  * Pre-request - Namaloda actual request execute aagurathuku munnadi ithu execute aagum
  * Post-response - Ithu nama request ku response vantha aparam execute aagum

Settings - namaku thevaiyana settings inga panalam
