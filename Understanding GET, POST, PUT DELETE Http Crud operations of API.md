SOAP - Heavy weight protocol

REST - Light weight protocol, have some leverages, it is easy to build and maintain, all the latest organizations like FB, Insta using REST

END POINT/Base URL: Address where API is hosted on the server

In REST there are some widely using methods like

  * GET: Oru information nama server la irunthu edukanum na intha method tha use pananum, server la irukura existing information.

  * POST: oru information ah nama server la ethurathukku ithu use panuvom, oru new information create pandrathukku

  * PUT: existing resource or data va change/update pandrathukku intha method use panuvom

  * DELETE: Ithu oru existing information or resource ah delete pandrathukku ithu use panuvom

Ithu major ah use pandrathunala itha nama CRUD nu solluvom (Create Retrive Update Delete)
